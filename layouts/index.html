What is Apache Ratis?

Apache Ratis is a higly customizable RAFT protocol implementation in java.

Pluggable transport

Transport layer of Ratis is separated from the RAFT implementation and can be used with any transport.

By default gRPC, Netty+Protobuf and Hadoop RPC based transports are provided.

Pluggable state machine

Ratis is a RAFT library which can use for many use cases. A custom state machine should be defined by the user of the library and Ratis will replicate the state between the servers.

Pluggable raft log

RAFT log is also pluggable, users may provide their own log implementation. The default implementation stores log in local files

Application could choose to not written all the data to log and state machine data and log data can be separately managed
